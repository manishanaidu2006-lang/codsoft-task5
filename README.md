A simple command-line Contact Book application built using Python that allows users to store, manage, and search contact information efficiently.

Features:-

 Add new contacts

 View all contacts

 Search contacts by name

 Update contact details

 Delete contacts

 Save contacts using file handling (optional)

User-friendly command-line interface
Usage:-

Run the application

Choose an option from the menu:

1 → Add Contact

2 → View Contacts

3 → Search Contact

4 → Update Contact

5 → Delete Contact

6 → Exit

Enter required details like name, phone number, or email when prompted.
